Crea Presentaciones Web
con SnapRoll.js

Una librería ligera y sin dependencias para construir experiencias inmersivas a pantalla completa con scroll de tipo "snap".

↓

Potente y Sencillo

  • Cero Dependencias

    Escrito en JavaScript puro (Vanilla JS). Ligero y rápido.

  • Secciones y Slides

    Soporta scroll vertical y slides horizontales dentro de cada sección.

  • Altamente Configurable

    Personaliza animaciones, selectores y comportamiento vía JS o HTML.

  • Enrutamiento por Hash

    Genera URLs únicas para cada sección y slide, ideal para compartir.

Empezar es Fácil

Paso 1: Incluye los archivos CDN

La forma más sencilla es vincular el CSS y el JS desde un CDN en tu archivo HTML. 

<!-- En tu <head> -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/snaprolljs/dist/snaprolljs.min.css" />

<!-- Antes de cerrar tu <body> -->
<script src="https://cdn.jsdelivr.net/npm/snaprolljs/dist/snaprolljs.min.js"></script>

Empezar es Fácil

Paso 2: Estructura tu HTML

Crea un contenedor principal (con la clase .sr-cont) y dentro, tus secciones (con la clase .sr-sec). 

<div class="sr-cont">
  <section class="sr-sec">Sección 1</section>
  <section class="sr-sec">Sección 2</section>
  <section class="sr-sec">Sección 3</section>
</div>

Empezar es Fácil

Paso 3: Activa SnapRoll

Añade el atributo data-snaproll a tu contenedor para la inicialización automática. ¡Y listo! 

<div class="sr-cont" data-snaproll>
  ...
</div>

Opciones de Configuración

Personaliza tu presentación con estas opciones, ya sea en el objeto JS o con atributos data-*. Desliza para ver más.

Opciones Generales

container '.sr-cont' Selector del contenedor principal.
loop false Navegación infinita entre secciones.
keyboard true Habilita navegación por teclado.
sectionAnimation 'slide' Animación ('slide', 'fade', 'zoom', etc.).

Opciones de Paginación

pagination true Muestra los puntos de navegación de sección.
paginationPosition 'right' Posición de los puntos ('right', 'left', 'top', 'bottom').
slideArrows true Muestra flechas en secciones con slides.
slidePagination true Muestra los puntos de navegación para slides.

Estructura HTML Avanzada

Usa atributos data-sr-* para configurar hashes, títulos y animaciones por sección o slide. 

<div class="sr-cont" data-snaproll data-section-animation="zoom">

  <!-- Configuración a nivel de sección -->
  <section class="sr-sec" 
           data-sr-hash="Inicio" 
           data-sr-title="Página Principal">
    ...
  </section>

  <section class="sr-sec" data-sr-hash="Servicios" data-sr-slide-animation="flip">
    <!-- Contenedor para slides horizontales -->
    <div class="sr-slides-cont">
      <div class="sr-slide" data-sr-hash="Diseño">Slide 1</div>
      <div class="sr-slide" data-sr-hash="Desarrollo">Slide 2</div>
    </div>
  </section>

</div>

API Pública

Controla la navegación mediante programación. Primero, obtén la instancia: 

// La instancia se puede encontrar en el elemento del contenedor
const miContenedor = document.querySelector('.sr-cont');
const miPresentacion = miContenedor.__snaproll_instance;

// Ahora puedes usar los métodos
miPresentacion.next();
Método Descripción
next() Navega al siguiente slide o sección.
prev() Navega al slide o sección anterior.
goToSection(index) Va a una sección específica por su índice.
goToSlide(index) Va a un slide específico en la sección actual.
refresh() Reinicializa (útil para contenido dinámico).
destroy() Elimina la instancia y todos los listeners.
